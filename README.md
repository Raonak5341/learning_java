#### What is Loop?
A loop statement allows us to execute a statement or group of statements multiple times.

<br/>

##### for loop syntax
---------------------------
for(initialization; condition; increment/decrement){
    statements(s)
}

#### while loop syntax
--------------------------
initialization;
while(condition) {
    statement(s)
    increment/decrement
}
